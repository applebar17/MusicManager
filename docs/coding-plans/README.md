# Coding Plans

This section turns the product requirements into implementation waves.

Current plans:

- [Backend Multiwave Plan](backend-multiwave-plan.md) - backend-first roadmap from
  local persistence to SoundCloud ingestion, matching, and export planning.
- [Frontend Multiwave Plan](frontend-multiwave-plan.md) - Tauri/TypeScript roadmap
  from desktop shell to the first complete workflow UI.
- [Shared Library Source Of Truth Plan](shared-library-source-of-truth-plan.md) -
  multi-wave roadmap for introducing a global flat library, USB alignment, metadata
  preservation, SoundCloud-to-library mapping, and library-backed USB exports.

The plans are intentionally staged. Each wave should leave the project in a runnable,
testable state before the next wave starts.
